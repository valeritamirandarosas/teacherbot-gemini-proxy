Endpoints del Proxy

GET /api/health

Estado del servicio.

curl -s https://TU-DOMINIO.vercel.app/api/health

POST /api/gemini

Proxy a Google Gemini (v1beta).

curl -X POST https://TU-DOMINIO.vercel.app/api/gemini \
 -H "Content-Type: application/json" \
 -d '{"prompt":"Hola, Gemini"}'

Privacidad y límites

Orígenes permitidos via ALLOWED_ORIGINS. Datos usados solo con fines educativos.
